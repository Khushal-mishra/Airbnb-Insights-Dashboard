🏡 Airbnb Property Insights Dashboard
1. Project Title / Headline
🏡 Airbnb Property Insights Dashboard

A modern, interactive Power BI dashboard designed to analyze Airbnb listings, market trends, guest ratings, and review behavior across major cities—helping uncover insights into property performance, host trust, and customer satisfaction.

2. Short Description / Purpose

The Airbnb Property Insights Dashboard provides a comprehensive view of Airbnb marketplace performance by combining listing growth, customer ratings, host credibility, and review analytics into one interactive report. It enables users to compare cities, identify market trends, evaluate customer satisfaction, and understand factors influencing booking behavior, making it valuable for hosts, investors, tourism analysts, and property management businesses.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Primary platform used for building interactive dashboards and reports.
📂 Power Query – Used for cleaning, transforming, and preparing Airbnb listing and review data.
🧠 DAX (Data Analysis Expressions) – Created dynamic KPIs, calculated measures, and conditional insights.
🔗 Data Modeling – Established relationships between listings, reviews, ratings, hosts, and city datasets for seamless filtering and analysis.
📈 Interactive Visualizations – Line charts, stacked column charts, bar charts, KPI cards, slicers, and custom infographic visuals.
📁 File Format – .pbix/.pbit for development and .png for dashboard previews.
4. Data Source

Source: Public Airbnb Open Data Dataset.

The dataset contains Airbnb property listings from multiple global cities, including:

Property information
Host details
Ratings
Reviews
Monthly review frequency
Trust indicators
Availability
Pricing
City-wise listing statistics

The dashboard integrates these datasets into a single analytical model for interactive exploration.

5. Features / Highlights
• Business Problem

The short-term rental market has become increasingly competitive, making it difficult for property owners, investors, and tourism analysts to quickly understand:

Which cities dominate the Airbnb market?
How have listings evolved over time?
Which properties receive the highest ratings?
How trustworthy are hosts?
What factors influence guest reviews?
How does customer engagement vary across seasons?

Raw Airbnb datasets contain thousands of records, making these insights difficult to uncover without visualization.

• Goal of the Dashboard

The dashboard is designed to:

Monitor Airbnb market growth across cities.
Analyze customer satisfaction using ratings.
Evaluate host credibility and trust indicators.
Identify review trends and seasonal booking behavior.
Support data-driven decisions for hosts, investors, and tourism businesses.
